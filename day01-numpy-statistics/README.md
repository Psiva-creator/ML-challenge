# Day 01 - NumPy Statistics

## 📌 Concept
NumPy arrays and basic statistical operations.

## 🛠️ What I Built
A random dataset statistics generator using NumPy.

## 📊 What It Does
- Generates a random dataset of 20 numbers
- Calculates mean, median, standard deviation, min, max, and range

## 🧠 What I Learned
- Creating NumPy arrays with `np.random.randint`
- Statistical functions: `mean`, `median`, `std`, `min`, `max`

## ▶️ How to Run
```bash
python project.ipynb
```
