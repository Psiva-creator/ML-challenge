# Day 02 - NumPy Operations

## 📌 Concept
NumPy array math operations and indexing.

## 🛠️ What I Built
A temperature data analyzer that processes weekly temperatures.

## 📊 What It Does
- Stores weekly temperature data
- Finds hottest and coldest days using argmax/argmin
- Converts Celsius to Fahrenheit using array math

## 🧠 What I Learned
- Array arithmetic operations
- `argmax`, `argmin` for finding index of extremes
- Broadcasting in NumPy

## ▶️ How to Run
```bash
python project.py
```
