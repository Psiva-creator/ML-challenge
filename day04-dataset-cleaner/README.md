# Day 04 - Dataset Cleaner

## 📌 Concept
Handling missing values in Pandas DataFrames.

## 🛠️ What I Built
A dataset cleaner tool that detects and fills missing values.

## 🧠 What I Learned
- Detecting nulls with `isnull().sum()`
- Filling missing values with `fillna()`
- Using mean and median for numeric imputation

## ▶️ How to Run
```bash
python project.py
```
