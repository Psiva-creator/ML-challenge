# Day 30 - Final Project: Cancer Detection Classifier

## 🎯 Project Goal
Build a complete ML pipeline to detect breast cancer as malignant or benign.

## 📌 What I Built
A model comparison system that trains 3 ML algorithms, evaluates them, and picks the best.

## 🛠️ Models Compared
- Logistic Regression (with StandardScaler pipeline)
- Random Forest Classifier
- Gradient Boosting Classifier

## 📊 Evaluation
- Test accuracy and 5-fold cross-validation score for each model
- Full classification report with precision, recall, F1-score

## 🧠 What I Learned Over 30 Days
- NumPy and Pandas for data manipulation
- Matplotlib for visualization
- Scikit-learn for ML: regression, classification, clustering, NLP
- Model evaluation, pipelines, hyperparameter tuning
- End-to-end ML project workflow

## ▶️ How to Run
```bash
python project.py
```
